Whole things is proggramed in python 3.14 using following libraries pyautogui, os. 
This repositary is made to scroll in websties, games, etc using just your keys.
Its made for somone to scroll in things that dont provide scroll bar but have a broken mouse scroll or just mouseless.
Instructions:                                                                                                                                                                                                                 (=) - scroll down                                                                                                                                                                                                            (-) - scroll up end                                                                                                                                                                                                             (end) - close the program


() is there for github to not break this file and display it weirdly and the end is just the end key
If u have any tips, ideas or things u want changed for a smaller keyboard tell me.
